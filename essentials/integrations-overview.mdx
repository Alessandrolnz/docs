# Integrations Overview

## Key Challenges Solved

Calmo integrates with your existing tools to solve critical challenges:

- **Overwhelming Observability** - Cut through alert noise and correlate signals across systems
- **Costly Downtime** - Reduce MTTR with faster detection and resolution workflows
- **Investigation Complexity** - Centralize relevant data to simplify root cause analysis
- **Knowledge Silos** - Capture context and solutions to prevent recurring incidents


## Getting Started

Each integration has its own setup process. Select a specific integration from the sidebar to learn how to configure and use it with Calmo.
